Sobre o projeto

Uma jornada diária para viver como filho, não como escravo.

👑 Propósito

O projeto De Escravo a Herdeiro nasceu para ajudar pessoas a fortalecerem sua identidade em Cristo por meio de afirmações bíblicas, oração, reflexão pessoal e desafios práticos.

✨ Como funciona

A jornada é formada por 365 etapas. A pessoa pode caminhar no próprio ritmo, confirmando o foco do dia, escrevendo reflexões e acompanhando seu progresso.

📖 Base espiritual

O conteúdo foi pensado para encorajar uma vida de fé, renovação da mente, constância e crescimento espiritual, sempre apontando para a identidade de filho em Cristo.

🤍 Observação importante

Esta jornada é um apoio devocional e espiritual. Ela não substitui aconselhamento pastoral, acompanhamento profissional ou cuidados médicos quando forem necessários.

Política de privacidade

Transparência sobre os dados da sua jornada.

🔒 Dados salvos no aparelho

Este site não exige login, cadastro, senha ou envio de informações pessoais. Seu progresso, confirmações e reflexões ficam salvos somente no navegador do aparelho usado, por meio do armazenamento local.

📝 Reflexões pessoais

As reflexões escritas dentro da jornada não são enviadas para servidor, banco de dados ou terceiros. Elas permanecem no próprio dispositivo enquanto o navegador mantiver esses dados salvos.

🧹 Limpeza dos dados

Se você limpar o histórico/dados do navegador ou usar a opção de limpar histórico dentro do site, o progresso e as reflexões poderão ser apagados.

📲 Compartilhamento

O botão de compartilhar/copiar usa recursos do próprio navegador. O conteúdo só é compartilhado quando você escolhe fazer isso.

💾 Backup

A opção de exportar backup gera um arquivo com os dados da jornada para você guardar. Esse arquivo fica sob sua responsabilidade depois de baixado.

🌐 Hospedagem

O site pode ser hospedado em uma plataforma de páginas estáticas, como GitHub Pages. A hospedagem entrega o arquivo do site, mas a jornada continua funcionando localmente no navegador.
